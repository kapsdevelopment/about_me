# Ken Andersen | levende webpresentasjon

En statisk nettside som oppfører seg som en klikkbar presentasjon med 13 slides.

Åpne `index.html` direkte i nettleseren, eller kjør en enkel lokal server:

```bash
python3 -m http.server 8124
```

Navigasjon:

- Klikk hvor som helst for neste slide.
- Bruk piltaster for forrige/neste.
- På mobil kan du sveipe sidelengs.

Eksterne logoer/key-art som er lastet ned lokalt ligger dokumentert i `ASSET_SOURCES.md`.
